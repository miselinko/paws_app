# Paws — Web Frontend

React 18 + TypeScript + Vite + Tailwind CSS web application.

See the [root README](../README.md) for full project documentation, setup instructions, and API reference.

## Dev

```bash
npm install
npm run dev        # http://localhost:5173
npm run build
npm run preview
```

Requires `VITE_API_URL` in `.env` pointing to the backend.
